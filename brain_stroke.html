¶

Un Brain Stroke (ou accident vasculaire crbral en franais) est larrt brutal de la circulation sanguine au niveau dune partie du cerveau).
Il y a deux types d'AVC :

  • Les AVC ischmiques qui sont dus un caillot dans une artre.
  • Les AVC hmorragiques qui sont dus une hmorragie, par lsions des parois du vaisseau.

Le but de ce jeu de donnes est de prdire si un patient eu ou non un AVC et pour cela, le jeu de donnes nous fournit 11 variables qui sont les
suivantes :

  • gender : genre du patient "Male" ou "Female".
  • age : l'ge du patient.
  • hypertension : variable binaire : 0 si le patient n'a pas d'hypertension, 1 si le patient a de l'hypertension.
  • heartdisease : variable binaire : 0 si le patient n'a pas de maladie cardiaque , 1 si le patient a une maladie cardiaque.
  • evermarried : "Yes" si le patient est marie, "No" sinon.
  • worktype : Type de travail : "children", "Govtjov", "Neverworked", "Private" ou "Self-employed".
  • Residencetype : Lieu de vie "Rural" ou "Urban".
  • avgglucoselevel : taux de glucose dans le sang en mg/dl.
  • bmi : L'indice de masse corporelle en kg/m$^2$.
  • smoking status : le statut de fumeur du patient : "formerly smoked", "never smoked", "smokes" ou "Unknown".
  • stroke : variable target : 1 si le patient a eu un AVC, 0 sinon.

Nous allons prsent calculer les diffrents scores de performances de chaque modles : l'Accuracy, le Recall, la Precison ainsi que le F1-Score.
Ces mtriques sont donnes par les formules suivantes :

$$Accuracy = \frac{Nombre \ de \ prdictions \ correctes}{Nombre \ de \ prdictions \ total} \ ; \ Recall = \frac{Vrai \ Positif}{Vrai \ Positif \ + \ Faux \ Ngatif} $$


$$ Precision = \frac{Vrai \ Positif}{Vrai \ Positif \ + \ Faux \ Positif} \ ; \ F1\_Score = 2 \ * \ \frac{Recall \ * \ Precision}{Recall \ + \ Precision} $$
